# LiDAR-
Arduino-CPP and Python codes for LiDAR &amp; Arduino UNO. 
ru: Код для настройки установки и отладки компонент устройства, таких как лазерный дальномер VL53L0X, шаговый двигатель 17HS4401 (+его драйвер A4988 и модуль-щит расширения CNC Shield V3.0) и мозг всея схемы Arduino UNO. Также код для получения и обработки 2D карты помещения в радиусе 1,3 метра (если возьмёте лазерный дальномер дороже - получите более подающее надежды устройство).
